# Dependency Hygiene CI — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a dedicated `dep-check` GitHub Actions job that posts an advisory-only PR comment summarising added/updated/removed packages, their licenses, npm download counts, and any CVEs scoped to changed packages.

**Architecture:** A separate workflow file (`.github/workflows/dep-check.yml`) triggered only when `package.json` or `pnpm-lock.yaml` changes. Analysis logic lives in `.github/scripts/dep-check.mjs` (plain Node 20 ESM, no extra deps). A `github-script` step reads its JSON output and posts/updates a single bot comment.

**Tech Stack:** GitHub Actions, Node 20 (native `fetch`, `spawnSync`), `pnpm audit --json`, npm registry REST API (`registry.npmjs.org`, `api.npmjs.org`), `actions/github-script@v7`

### Task 1: Create the analysis script

**Files:**
- Create: `.github/scripts/dep-check.mjs`

**Step 1: Create the directory and file**

```bash

```

Create `.github/scripts/dep-check.mjs`:

```js
#!/usr/bin/env node
// Dep-check analysis script — outputs JSON to stdout.
// Environment variables consumed:
//   BASE_REF  — base branch name (e.g. "main")
//
// Exit 0 always (advisory only).

import { spawnSync } from 'node:child_process'

function spawn(cmd, args) {
  const result = spawnSync(cmd, args, { encoding: 'utf8' })
  if (result.error) throw result.error
  return result.stdout ?? ''
}

// --- 1. Diff the lockfile ---
const baseRef = process.env.BASE_REF ?? 'main'
let diffOutput = ''
try {
  diffOutput = spawn('git', ['diff', `origin/${baseRef}...HEAD`, '--', 'pnpm-lock.yaml'])
} catch {
  console.log(JSON.stringify({ added: [], updated: [], removed: [], audit: [] }))
  process.exit(0)
}

// Parse package entry lines: `+  'pkg@version':` or `+  pkg@version:`
// pnpm lockfile v9 format: each package is a YAML key like `  'name@version':`
const ENTRY_RE = /^([+-])\s{2,}['"]?(@?[\w\-./@]+@[\d\w\-^~>=.<*|, ]+?)['"]?:\s*$/

const plusPkgs = new Map()   // name → version
const minusPkgs = new Map()

for (const line of diffOutput.split('\n')) {
  const m = line.match(ENTRY_RE)
  if (!m) continue
  const sign = m[1]
  const full = m[2].trim()
  // Split on last @ to separate scoped package names from version
  const atIdx = full.lastIndexOf('@')
  if (atIdx <= 0) continue
  const name = full.slice(0, atIdx)
  const version = full.slice(atIdx + 1)
  if (sign === '+') plusPkgs.set(name, version)
  else minusPkgs.set(name, version)
}

const added = []
const updated = []
const removed = []

for (const [name, version] of plusPkgs) {
  if (minusPkgs.has(name)) {
    updated.push({ name, from: minusPkgs.get(name), to: version })
  } else {
    added.push({ name, version })
  }
}
for (const [name, version] of minusPkgs) {
  if (!plusPkgs.has(name)) removed.push({ name, version })
}

// --- 2. npm metadata for added packages ---
const RISKY_LICENSES = new Set([
  'GPL-2.0', 'GPL-3.0', 'AGPL-3.0',
  'LGPL-2.0', 'LGPL-2.1', 'LGPL-3.0',
  'UNLICENSED',
])

async function fetchMeta(name) {
  try {
    const encoded = encodeURIComponent(name)
    const [reg, dl] = await Promise.all([
      fetch(`https://registry.npmjs.org/${encoded}/latest`).then(r => r.json()),
      fetch(`https://api.npmjs.org/downloads/point/last-week/${encoded}`).then(r => r.json()),
    ])
    const license = reg.license ?? 'unknown'
    return {
      license,
      licenseOk: !RISKY_LICENSES.has(license) && license !== 'unknown',
      downloads: dl.downloads ?? null,
      deprecated: reg.deprecated ?? null,
    }
  } catch {
    return { license: 'unknown', licenseOk: false, downloads: null, deprecated: null }
  }
}

const metaResults = await Promise.all(added.map(p => fetchMeta(p.name)))
const addedWithMeta = added.map((p, i) => ({ ...p, ...metaResults[i] }))

// --- 3. pnpm audit (scoped to changed packages) ---
const changedNames = new Set([
  ...added.map(p => p.name),
  ...updated.map(p => p.name),
])

let audit = []
try {
  const raw = spawnSync('pnpm', ['audit', '--json'], { encoding: 'utf8' }).stdout ?? ''
  const parsed = JSON.parse(raw)
  // pnpm audit JSON: { advisories: { [id]: { module_name, severity, title, url } } }
  const advisories = parsed.advisories ?? {}
  audit = Object.values(advisories)
    .filter(a => changedNames.has(a.module_name))
    .map(a => ({ name: a.module_name, severity: a.severity, title: a.title, url: a.url }))
} catch {
  // audit unavailable or no advisories — leave audit as []
}

console.log(JSON.stringify({ added: addedWithMeta, updated, removed, audit }))
```

### Task 2: Create the workflow

**Files:**
- Create: `.github/workflows/dep-check.yml`

**Step 1: Write the workflow**

```yaml
name: Dep Check

on:
  pull_request:
    branches: [main]
    paths:
      - '**/package.json'
      - '**/pnpm-lock.yaml'

concurrency:
  group: dep-check-${{ github.ref }}
  cancel-in-progress: true

jobs:
  dep-check:
    name: Dependency Hygiene
    runs-on: ubuntu-latest
    permissions:
      pull-requests: write

    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: pnpm/action-setup@v4

      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: pnpm

      - run: pnpm install --frozen-lockfile

      - name: Run dep-check analysis
        id: analysis
        run: |
          RESULT=$(BASE_REF=${{ github.base_ref }} node .github/scripts/dep-check.mjs)
          echo "result<<EOF" >> $GITHUB_OUTPUT
          echo "$RESULT" >> $GITHUB_OUTPUT
          echo "EOF" >> $GITHUB_OUTPUT

      - name: Post dep-check comment
        uses: actions/github-script@v7
        env:
          ANALYSIS: ${{ steps.analysis.outputs.result }}
        with:
          script: |
            const data = JSON.parse(process.env.ANALYSIS)
            const { added, updated, removed, audit } = data

            const fmtDownloads = (n) => {
              if (n == null) return 'unknown'
              if (n >= 1_000_000) return `${(n / 1_000_000).toFixed(1)}M`
              if (n >= 1_000) return `${(n / 1_000).toFixed(1)}k`
              return String(n)
            }

            const sections = []

            if (added.length) {
              const rows = added.map(p => {
                const lic = `${p.license} ${p.licenseOk ? '✅' : '⚠️'}`
                const dep = p.deprecated ? ` ⚠️ *deprecated: ${p.deprecated}*` : ''
                return `| \`${p.name}\` | ${p.version} | ${lic} | ${fmtDownloads(p.downloads)} |${dep}`
              }).join('\n')
              sections.push(`### Added (${added.length})\n| Package | Version | License | Downloads/wk |\n|---------|---------|---------|-------------|\n${rows}`)
            }

            if (updated.length) {
              const rows = updated.map(p => `| \`${p.name}\` | ${p.from} | ${p.to} |`).join('\n')
              sections.push(`### Updated (${updated.length})\n| Package | From | To |\n|---------|------|----|  \n${rows}`)
            }

            if (removed.length) {
              const list = removed.map(p => `- \`${p.name}\` (${p.version})`).join('\n')
              sections.push(`### Removed (${removed.length})\n${list}`)
            }

            if (!added.length && !updated.length && !removed.length) {
              sections.push('_No dependency changes detected in lockfile._')
            }

            const auditSection = audit.length
              ? `### 🔐 Audit\n${audit.map(a => `- **${a.severity}** [\`${a.name}\`](${a.url}): ${a.title}`).join('\n')}`
              : `### 🔐 Audit\nNo vulnerabilities found in changed packages.`
            sections.push(auditSection)

            const body = [
              '## 📦 Dependency Changes',
              '',
              ...sections,
              '',
              `> Run [#${context.runNumber}](${context.serverUrl}/${context.repo.owner}/${context.repo.repo}/actions/runs/${context.runId})`,
            ].join('\n\n')

            const { data: comments } = await github.rest.issues.listComments({
              owner: context.repo.owner,
              repo: context.repo.repo,
              issue_number: context.issue.number,
            })
            const existing = comments.find(c => c.user.type === 'Bot' && c.body.startsWith('## 📦 Dependency Changes'))
            if (existing) {
              await github.rest.issues.updateComment({
                owner: context.repo.owner,
                repo: context.repo.repo,
                comment_id: existing.id,
                body,
              })
            } else {
              await github.rest.issues.createComment({
                owner: context.repo.owner,
                repo: context.repo.repo,
                issue_number: context.issue.number,
                body,
              })
            }
```

### Notes

- **No new runtime deps** — script uses only Node 20 builtins (`spawnSync`, native `fetch`).
- **Shell injection safety** — `spawnSync` takes argument arrays, never passes strings through a shell. `BASE_REF` comes from the trusted `github.base_ref` context.
- **Lockfile parse robustness** — regex targets pnpm v9's `name@version:` YAML key pattern. Empty diff → empty result, no crash.
- **Advisory only** — workflow never exits non-zero. `pnpm audit` non-zero status is ignored via `spawnSync` (status code not checked).
- **Rate limits** — npm registry public API has generous limits; parallel fetching a handful of new packages is safe.
